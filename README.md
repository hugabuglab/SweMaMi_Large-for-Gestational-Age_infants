# Maternal Gut Microbiome and Large-for-Gestational-Age Birth (SweMaMi)

Analysis code accompanying the manuscript:

> **Maternal gut microbiome diversity and functional potential in early pregnancy are associated with large-for-gestational-age birth: a nested case-control study within the SweMaMi cohort**
> Antony AT, Mohseni F, Toubon G, Guðnadóttir U, Engstrand L, Fransson E, Schuppe-Koistinen I, Brusselaers N, Hugerth LW.

This repository documents the full analysis pipeline used in the study. It is provided for **transparency**. Note that the raw sequencing data are not hosted here (see [Data availability](#data-availability)),the analyses cannot be re-run end-to-end from this repository alone, but every analytical step reported in the manuscript is included.

---

## Overview

This is a nested case-control study within the Swedish Maternal Microbiome (SweMaMi) cohort comparing the gut microbiome of mothers delivering large-for-gestational-age (LGA) versus appropriate-for-gestational-age (AGA) infants, at two gestational timepoints (TP1: weeks 10–20; TP2: weeks 28–32), using shotgun metagenomics.

## Analysis workflow

| Notebook | Analysis | Manuscript output |
|----------|----------|-------------------|
| `01_data_preparation` | 1:2 propensity score matching (MatchIt), species filtering (≥10⁻⁵ in ≥5% of samples), confounder definitions | Figure 1, Table 1 |
| `02_alpha_diversity` | Species richness, Shannon, Pielou's evenness; t-tests, logistic regression, linear mixed-effects models for longitudinal change | Figures 2, 3 |
| `03_beta_diversity` | Bray-Curtis, Jaccard, Aitchison distances; betadisper, PERMANOVA, random subsampling | Figure 4 |
| `04_differential_abundance` | MaAsLin3 differential abundance/prevalence at TP1 | Figure 5 |
| `05_gut_metabolic_modules` | Gut Metabolic Module profiling (MaAsLin3 + logistic regression) at TP1 and TP2 | Figures 6, 7 |
| `06_species_contributions` | KO gene family contributions of *Anaerostipes hadrus* to AGA-associated modules | Figure 9 

**Note on Figure 8:** The gut metabolic map (Figure 8) is a conceptual figure
created manually (adapted from Vieira-Silva et al.) to illustrate the positions
of the differentially associated modules identified in this analysis. It is not
generated by code and therefore has no corresponding notebook.

## Sensitivity analysis

Sensitivity analyses were performed by restricting the cohort to participants
without gestational diabetes mellitus (GDM) or excessive gestational weight gain
(EGWG), using the same analytical methods as the primary analysis (alpha diversity,
beta diversity,differential abundance and gut metabolic modules). As matching could not be maintained
after exclusion, these analyses were unmatched. Full sensitivity results are
provided in Supplementary Data 5–6 and Supplementary Tables S4–S5.

## Software and environment

Analyses were performed in **R (v4.5.2)** within Jupyter notebooks (R kernel). Key packages:

- `tidyverse` (v2.0.0), `dplyr` (v1.2.0)
- `ggplot2` (v4.0.2), `ggrain` (v0.0.4)
- `vegan` (v2.7.3)
- `maaslin3` (v0.99.16)
- `lme4` (v1.1.33)
- `MatchIt`

Upstream bioinformatic processing used the StaG-mwc (v0.5.1) workflow (FastP, Kraken2, MetaPhlAn4, HUMAnN2, Omixer-RPM). See the manuscript Methods for full details.

## Data availability

The raw metagenomic sequences, along with selected metadata variables, are deposited in the European Nucleotide Archive (ENA) under project number **PRJEB81814**.

Processed abundance tables (MetaPhlAn4 species profiles and Gut Metabolic Module estimates) are provided as Supplementary Data with the manuscript.

Individual-level participant data are restricted for privacy reasons and are not included in this repository. See `data/README.md` for details.
